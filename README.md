## ğŸš¦ Getting Started

This is a **static website** - no build process required!

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/gab-cat/gab-portfolio-v2.git
   cd gab-portfolio-v2
   ```

2. **Open in browser**
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   
   # Or simply open index.html in your browser
   ```
